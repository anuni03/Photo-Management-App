# Photo Management App

This Photo Management App is a React-based web application that allows users to upload, view, and manage photos. It utilizes Firebase for data storage and real-time updates.

## Features

- Photo Upload: Users can upload photos along with titles and descriptions.
- Real-Time Gallery: The app displays photos in real-time as they are added.
- Firebase Integration: Leveraging Firebase Firestore for data storage and Firebase Storage for image hosting.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.
Need to have Node version 17+


### Prerequisites

- Node.js
- npm (Node Package Manager)
- Firebase account

### Installing

1. Clone the repository

   bash
   git clone [repository URL]
   cd Photo-Management-App
   

2. Install dependencies

   bash
   npm install
   

3. Set up Firebase
   
   - Create a Firebase project in the Firebase console.
   - Add Firebase configuration to your project.

4. Start the development server

   bash
   npm start
   

   The app should now be running on `http://localhost:3000`.

## Built With

- [Firebase](https://firebase.google.com/) - Database and file storage
